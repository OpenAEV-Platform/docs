# OpenAEV SCV

!!! info ""

    * **Deprecated in**: `OpenAEV 2.0.0`
    * **Removed in**: `OpenAEV 2.1.0`

## Description of changes

A lot of processes have been automated, but there are still small modifications to do manually.

**OpenAEV Collectors/Injectors **

All our Docker images have been updated to include the SCV domains values. If you're using one of them, be careful to load the latest version from our [Docker repository](https://hub.docker.com/search?q=openaev)


## Migration guide

**OpenAEV SCV**

- If you're using our official [docker compose](https://github.com/OpenAEV-Platform/docker), be sure to update your collectors/injectors images from the OpenAEV Docker repository.
- If you're not using our official [docker compose](https://github.com/OpenAEV-Platform/docker), be sure to update your collectors/injectors images from the [Docker repository](https://hub.docker.com/search?q=openaev).
